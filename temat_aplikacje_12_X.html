CSS

Aplikacje i strony internetowe

Niedawno poznaliśmy CSS'a, czyli kaskadowe arkusze stylów. CSS pomaga nam w poprawie wyglądu naszej strony.
CSS podpinamy do pliku HTML w sekcji head za pomocą tagu link np.: < link rel="stylesheet" href="style.css" >
W CSS'ie struktura wygląda następująco:

NAZWA TAGU {
ATRYBUT: WARTOŚĆ;
}
.NAZWA ID {
ATRYBUT: WARTOŚĆ;
}
#NAZWA KLASY {
ATRYBUT: WARTOŚĆ;
}

 Na przykład:

body {
background-color: blue;
}
.reklama {
width: 100px; height: 200px;
}
#tresc {
color: red;
}

Przetestuj to sam!
